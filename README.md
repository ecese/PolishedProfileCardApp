## Project Overview

ProfileCardApp is a React Native project built with Expo. The app demonstrates advanced styling techniques, including custom fonts, theme support (light/dark), responsive layouts, shadows, and interactive components like Like and Follow buttons.

This project was developed as part of **Mobile Programming - Lab 09: React Native Lab 4: Styling & Polish**.

---

## Features

- **Custom Fonts:** Montserrat Regular and Bold loaded using `expo-font`.  
- **Theme Support:** Light and dark mode toggle.  
- **Responsive Layout:** Adjusts card size and icon based on screen width.  
- **Profile Card:** Polished card with shadow and rounded corners.  
- **Interactive Buttons:**  
  - **Like Button:** Changes color on press and logs "Profile Liked!" to console.  
  - **Follow Button:** Toggles between Follow and Following states.
